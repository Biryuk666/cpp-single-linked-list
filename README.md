# Односвязный список

## Описание
**Односвязный список** - это структура данных на базе C++, которая выполняет базовый функционал *std::forward_list*, но при этом итоговое приложение с ней занимает меньший объем памяти, что в условиях ограниченной памяти может быть критически важно. Поскольку **односвязный список** работает с динамической памятью, то для предотвращения утечек, в нем присутствует собственный сборщик мусора.

## Возможности
* Метод *GetSize()* возвращает размер контейнера.
* Метод *IsEmpty()* сообщает, пустой ли контейнер.
* Метод *Clear()* удаляет все элементы контейнера и обнуляет его размер.
* Метод *PushFront(...)* добавляет указанный элемент в начало контейнера.
* Метод *PopFront()* удаляет первый элемент контейнера.
* Метод *InsertAfter(...)* добавляет элемент в слудующую за указанной позицией контейнера. Возвращаемым значением метода является итератор (указатель) на вставленный элемент. Если по какой-либо причине вставить элемент в контейнер не удастся, контейнер останется в прежнем состоянии.
* Метод *EraseAfter(...)* удаляет элемент в следующей за указанной позицией контейнера и возвращает итератор (указатель) на следующий за удаленным элемент.
* Метод *Swap(...)* обменивает значения с указанным контейнером.
* Методы *before_begin()* и *cbefore_begin()* возвращают итератор (указатель) и константный итератор на позицию перед первым элементом контейнера. Данные итераторы запрещено разыменовывать, поскольку это приведет к неопределенному поведению.
* Методы *begin()* и *cbegin()* возвращают итератор (указатель) и константный итератор на позицию первого элемента контейнера.
* Методы *end()* и *cend()* возвращают итератор (указатель) и константный итератор на следующую за последним элементом позицию. Данные итераторы запрещено разыменовывать, поскольку это приведет к неопределенному поведению.

## Использование
Для использования **односвязного списка** необходимо скопировать файл *single_linked_list.h* в каталог с проектом и импортировать его как стороннюю библиотеку.
